# star-wars
Projeto em Angular Star Wars!

Link: https://github.com/dbnck/star-wars

Escolha um diretório para baixar o projeto.

Abra um terminal, e utilizando git, faça um "git clone https://github.com/dbnck/star-wars.git".

Acesse o diretório do projeto (star-wars) e digite "npm install" para baixar as dependências do projeto.

Após baixar as dependências digite "npm start" para iniciar a aplicação.

Pronto, a aplicação está disponível em http://localhost:4200
